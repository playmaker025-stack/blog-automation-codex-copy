# Agent SEO Analyst Reference

## 네이버 SERP 모듈 기반 키워드 분류

### 적용 배경

네이버 키워드 판단은 더 이상 `블로그탭 있음/없음`만으로 결정하지 않는다. SEO Analyst는 모바일 통합검색 첫 화면에서 어떤 모듈이 먼저 노출되는지를 우선 확인하고, 그 결과를 전략 수립과 글쓰기 구조에 전달한다.

공식 확인된 변화:

- 2025년 3월 27일 네이버 AI 브리핑 전체 사용자 출시. 네이버 공식 보도자료 기준으로 공식형/멀티출처형, 숏텐츠형, 플레이스형, 쇼핑형 등 유형별 UX가 적용된다.  
  출처: https://navercorp.com/media/pressReleasesDetail?seq=32374
- 2026년 4월 28일 네이버 AI탭 베타 출시. 네이버플러스 멤버십 사용자를 대상으로 검색, 쇼핑, 로컬 등 버티컬 연결을 강화한 대화형 검색 경험을 제공한다.  
  출처: https://www.navercorp.com/media/pressReleasesDetail?seq=34984
- 2026년 5월 28일 전후 네이버 콘텐츠/AI 검색 전략 발표에서 AI탭 확대, AI 브리핑과 콘텐츠 노출 강화 흐름이 확인된다.  
  출처: https://www.chosun.com/economy/tech_it/2026/05/28/VJXI7J356VGSVGPOCGHHDXEJPQ/

운영 가설:

- 블로그탭 유무는 보조 신호다.
- 핵심 판단 기준은 모바일 통합검색 첫 화면의 첫 블록이다.
- 블로그 글은 블로그/뷰 영역뿐 아니라 AI 브리핑 인용, 플레이스 보조 문서, 클립 보조 텍스트로도 역할을 가질 수 있다.

### SERP 모듈 분류값

SEO Analyst는 키워드를 아래 5개 모듈 중 하나로 분류한다.

| serpModule | 첫 화면 신호 | 검색 의도 | 기본 대응 |
| --- | --- | --- | --- |
| `ai_briefing` | AI 브리핑 텍스트 요약 | 정보 탐색, 비교, 판단 | AEO 오프닝과 FAQ 중심 글 |
| `clip` | 클립/숏폼 영상 | 체감, 시연, 후기 | 클립 병행, 블로그는 보조 텍스트 |
| `place` | 지도, 매장, 플레이스 | 로컬 방문, 상담 | 스마트플레이스 보조 + 방문 전 안내 글 |
| `shopping` | 쇼핑 상품, 가격, 구매 버튼 | 구매 전환 | 쇼핑/스토어 우선, 블로그는 보조 |
| `blog_view` | 블로그, 카페, 포스트 문서 | 롱폼 정보, 문제 해결, 사용법 | 기존 DIA/C-Rank형 블로그 글 |

### 판정 순서

1. 모바일 네이버 통합검색 첫 블록을 확인한다.
2. 첫 블록을 `ai_briefing`, `clip`, `place`, `shopping`, `blog_view` 중 하나로 분류한다.
3. 직접 확인이 어려우면 키워드 언어 패턴으로 임시 예측한다.
4. 연관 검색어와 함께 많이 찾는 방향을 보조 신호로 본다.
5. 블로그탭/뷰탭 유무는 마지막 보조 신호로만 사용한다.

### 언어 패턴 기반 임시 예측

| 패턴 | 신호 키워드 | 예상 serpModule |
| --- | --- | --- |
| 정보/비교형 | 차이, 비교, 어떤, 뭐가 나을까, 이유, 정리 | `ai_briefing` |
| 체감/시연형 | 후기, 리뷰, 실사용, 언박싱, 맛, 연기량, 체감 | `clip` |
| 로컬 방문형 | 지역명 + 전자담배/베이프샵/매장/상담 | `place` |
| 구매 전환형 | 가격, 구매, 살 곳, 최저가, 제품명 단독 | `shopping` |
| 문제 해결형 | 고장, 탄맛, 누수, 안됨, 원인, 해결, 증상 | `blog_view` |
| 사용법/팁형 | 교체 시기, 설정, 사용법, 방법, 관리 | `blog_view` |

### SEO Analyst 출력 필드

SEO Analyst는 기존 검색의도 결과에 아래 필드를 추가한다.

```json
{
  "primarySearchIntent": "비교형",
  "serpModule": "ai_briefing",
  "serpModuleReason": "모바일 통합검색 첫 블록이 AI 브리핑이고 비교/판단형 질의입니다.",
  "serpModuleOptions": ["ai_briefing", "clip", "place", "shopping", "blog_view"],
  "recommendedBlogRole": "B",
  "contentType": "aeo_comparison",
  "aeoTarget": true,
  "blogTabIsPrimarySignal": false,
  "blogTabExists": false,
  "clipCompetition": false
}
```

필드 의미:

- `primarySearchIntent`: 정보형, 비교형, 방문형, 구매검토형, 문제해결형, 사용법형 등 실제 검색 의도
- `serpModule`: 최종 SERP 모듈 분류
- `serpModuleReason`: 판정 근거
- `recommendedBlogRole`: A/B/C/D/E 중 우선 배정 블로그
- `contentType`: writer가 사용할 글 구조 타입
- `blogTabIsPrimarySignal`: 항상 `false`를 기본값으로 둔다

### 블로그 역할 배정

| serpModule | 우선 블로그 | 이유 |
| --- | --- | --- |
| `ai_briefing` | B 또는 C | 비교/판단은 B, 구조/정보 설명은 C |
| `clip` | E | 후기, 체감, 실용 보조 텍스트 |
| `place` | A | 지역 허브, 방문 전 안내, 플레이스 보조 |
| `shopping` | A 또는 B | 구매 전환 보조, 스토어/매장 연결 |
| `blog_view` 문제해결 | D | 증상, 원인, 해결 중심 |
| `blog_view` 사용법 | C | 사용법, 설정, 구조 설명 |

### 모듈별 Writer 지침

#### `ai_briefing`

- 첫 문단에 직접 답변형 한 문장을 배치한다.
- 질문형 또는 구분형 제목을 우선한다.
- FAQ 2~3개를 하단에 둔다.
- `~란 무엇인가`, `완벽 정리`, `총정리` 같은 정의형 제목은 피한다.
- AI 브리핑이 본문을 그대로 대체하지 못하도록 실제 상담 맥락과 판단 기준을 넣는다.

#### `clip`

- 블로그 단독 상위노출을 목표로 잡지 않는다.
- 클립 영상 또는 시연 콘텐츠를 보조하는 설명 글로 쓴다.
- 체감, 후기, 사용 장면, 비교 포인트를 짧고 명확하게 정리한다.
- E블로그 실용/후기형 배정을 우선한다.

#### `place`

- 매장 위치, 주차, 영업시간, 방문 전 확인 사항을 포함한다.
- 스마트플레이스 CTA를 넣는다.
- 글 목표는 블로그 단독 상위노출보다 플레이스 보조 문서 노출이다.
- A블로그 지역 허브형 배정을 우선한다.

#### `shopping`

- 제품 가격/구매 전환 키워드는 쇼핑/스토어가 우선임을 전제로 한다.
- 블로그는 구매 전 확인 기준, 비교 기준, 매장 상담 연결을 보조한다.
- 단정적인 최저가 표현은 피한다.

#### `blog_view`

- 문제해결형은 증상, 원인, 해결 3단계 구조를 유지한다.
- 사용법형은 설정, 교체, 관리 기준을 단계별로 쓴다.
- 기존 DIA/C-Rank 기준, 실제 경험, 내부링크 구조를 함께 적용한다.

### 만수르베이프 전자담배 예시

전자담배 예시는 업종 전용 규칙이 아니라, 모듈 판정을 이해하기 위한 참고 사례다. 실제 구현은 다른 업종에도 적용 가능한 범용 로직으로 유지한다.

#### `ai_briefing` 후보

- 전자담배 입호흡 폐호흡 차이
- 전자담배 처음 살 때 뭐가 좋을까
- 액상 니코틴 농도 차이
- 입문자 전자담배 추천
- 전자담배 vs 궐련 차이
- 전자담배 액상 성분
- 코일 종류 차이
- 베이프 용어 정리

#### `clip` 후보

- 전자담배 연기량 비교
- 액상 맛 후기
- 전자담배 언박싱
- 제품명 + 후기

#### `place` 후보

- 인천 전자담배
- 부평 전자담배
- 만수동 전자담배
- 구월동 전자담배
- 부평 베이프샵
- 인천 전자담배 가게 추천
- 부평 전자담배 상담

#### `blog_view` 후보

- 전자담배 코일 탄맛 원인
- 전자담배 액상 누수 해결
- 전자담배 안켜짐 원인
- 전자담배 코일 교체 시기
- 전자담배 와트 설정 방법
- 전자담배 액상 분리현상
- 전자담배 충전 안됨
- 코일 번 방지법

### 탭 구분 보조 지식

| 탭 | 포함 콘텐츠 | 의미 |
| --- | --- | --- |
| 블로그탭 | 블로그 전용 | 블로그 단독 경쟁 구조 가능 |
| 뷰탭 | 블로그, 카페, 포스트 통합 | 블로그만 경쟁자가 아님 |
| 탭 없음 | AI 브리핑, 클립, 플레이스, 쇼핑 등이 점유 | 해당 모듈 전략으로 대응 |

### 빠른 판별 카드

```text
키워드 입력
  ↓
모바일 통합검색 첫 블록 확인
  ├─ AI 브리핑 → AEO 구조로 작성, B/C 블로그
  ├─ 클립 영상 → 클립 병행, E 블로그 보조
  ├─ 지도/플레이스 → 방문 전 안내 글, A 블로그
  ├─ 쇼핑 → 스토어/구매 전환 보조
  └─ 블로그/카페 문서 → DIA/C-Rank 기준, C/D 블로그
```
