" Malone Bar (말론바) "

라는 이름을 가지고

일회용 전담으로 출시가 되었습니다.

자세한 정보를 알려드리기 전에

간단한 스펙을 먼저 알려드리도록 할게요!

퍼프수 : 35,000퍼프

배터리 : 650mAh

충전 : C 타입

니코틴 함량 : 0.98%

코일 : 듀얼 메쉬

조절 기능 : 출력 / 밝기


위 사진으로 보이는 4가지의 맛 말고도

여러 가지의 종류가 출시되었지만

​

이번 글에서는 스펙과 사용법을

다룰 거라서 각각의 맛 후기는

따로 포스팅해서 다루도록 할게요!

​





패키징 박스의

정면과 후면에는

제품명과 기기 그림이 있고

측면에는

더 허전하게 느껴집니다.

말론바 기기의 하단에는

에어홀 조절바, 버튼, C 타입 충전 단자

이렇게 3가지가 보입니다.

​

흡입압 조절바와 버튼을 이용해서

기능을 사용하실 수 있어요!

이따 사용방법을 알려드릴 테니

천천히 따라해 보시면

쉽게 적응하실 수 있을 겁니다.
